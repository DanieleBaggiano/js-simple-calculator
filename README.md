ESERCIZIO CALCOLATRICE

# RACCOLTA DATI
- Numeri inseriti dall'utente tramite i pulsanti numerici
- Operatore selezionato dall'utente tramite i pulsanti operatore (+, -, *, /)
- Risultato calcolato dall'operazione selezionata

# ESECUZIONE LOGICA
- Aggiungere event listener a tutti i numeri
- Quando si clicca su un numero, lo concatena al numero visualizzato sopra
- Aggiungere event listener a tutti gli operatori
- Quando si clicca su un operatore, salvare l'operatore cliccato e il primo operando, poi resettare il numero visualizzato sopra
- Aggiungere event listener al pulsante uguale
- Quando si clicca su uguale, salvare il secondo operando, effettuare il calcolo corretto in base all'operatore selezionato e visualizzare il risultato sopra
- Quando clicco sul pulsante canc, cancellare il calcolo facendo comparire lo zero

# OUTPUT
- Il risultato delle operazioni viene visualizzato nel display della calcolatrice
